🚀 COMMANDES PRINCIPALES
=========================

1️⃣  Initialisation du projet:
   python main.py

2️⃣  Notebook d'EDA:
   jupyter notebook notebooks/01_eda.ipynb

3️⃣  Notebook de Modélisation:
   jupyter notebook notebooks/02_modeling.ipynb

4️⃣  Lancer le Dashboard:
   streamlit run app/dashboard.py

5️⃣  Lancer l'API (optionnel):
   uvicorn app.api:app --reload

📊 MODULES DISPONIBLES

✅ DataProcessor      - Chargement et exploration des données

✅ ModelEvaluator     - Évaluation des performances
✅ ModelComparator    - Comparaison multi-modèles
✅ ExplainabilityAnalyzer - Feature Importance & SHAP

✅ Dashboard Streamlit - Interface décisionnelle
✅ API FastAPI        - Service d'inférence
